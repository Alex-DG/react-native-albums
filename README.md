# react-native-albums
- estlint / airbnb config
- jest
- one entry point App.js for both platforms

### Setup

#### React Native CLI

```
$ npm install -g react-native-cli
```

#### Dependencies

```
$ npm install
```

### Commands

#### Run iOS

```
$ react-native run-ios
```

#### Run Android

```
$ react-native run-android
```
